# Deep Fake Detection using Pre-trained Models

This project presents a comparative performance analysis of multiple pre-trained deep learning models for deepfake detection.

It evaluates models on an image dataset (Real vs Fake) and compares them using standard classification metrics such as accuracy, precision, recall, F1-score, sensitivity, and specificity.

---

## Features

* Uses 10 pre-trained models:

  * ResNet18
  * MobileNetV2
  * EfficientNet-B0
  * VGG16
  * DenseNet121
  * AlexNet
  * SqueezeNet
  * ShuffleNet
  * ResNet34
  * GoogLeNet

* Applies fine-tuning (training only the final classification layer)

* Computes evaluation metrics:

  * Sensitivity
  * Specificity
  * Precision
  * Recall
  * F1 Score
  * Accuracy

* Generates:

  * Console output table
  * `final_table.png` (submission-ready comparison table)

## Methodology

1. Load dataset (Real vs Fake images)
2. Apply preprocessing (resize and tensor conversion)
3. Load pre-trained models with ImageNet weights
4. Replace final classification layer for binary classification
5. Fine-tune models on the dataset
6. Generate predictions
7. Evaluate using classification metrics

---

## Notes

* Only the final layer of each model is fine-tuned
* Minimum dataset size requirement (≥ 100 samples) is satisfied
* Results may vary depending on dataset quality and distribution
